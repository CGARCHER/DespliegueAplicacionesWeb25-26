Tarea: Desplegar la aplicación "Pets" en Render

Objetivo
- Desplegar en Render los tres componentes de la aplicación: base de datos PostgreSQL (servicio de Render), backend (API) y frontend, usando la rama feature-docker del repositorio.

Entregables
1. URL pública del frontend.
2. URL pública del backend (API).
3. Una captura de pantalla del panel de Render mostrando las variables de entorno del servicio (puede ocultarse la contraseña).
4. Informe con problemas encontrados y cómo se resolvieron.

Repositorio y rama
- Repo: https://github.com/CGARCHER/pets-app
- Rama: feature-docker
- Puedes usar tu propio proyecto si ya lo tienes.

Criterios de evaluació
- Frontend accesible y consumiendo la API.
- Backend funcionando y accesible.
- Base de datos PostgreSQL creada y utilizada por la aplicación.
- Evidencia y pequeño informe entregados.
